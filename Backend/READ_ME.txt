# Backend

## Setup

1. Install Node.js:
https://nodejs.org/en/download

2. Install Dependencies. Run this:
```sh
npm install
```

3. Test. Run this:
```sh
npm run dev
```

## Scripts

This will run the backend.
```sh
npm run dev
```

This will format the code.
```sh
npm run format
```

This will check for code quality.
```sh
npm run lint
```
